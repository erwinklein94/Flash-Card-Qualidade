# Atualização visual Rumo com zoom out

Este pacote recebeu o mesmo padrão visual aplicado nos demais sistemas Rumo do Hub de Qualidade.

## O que foi alterado

- Aplicação da paleta visual Rumo: azul institucional, azul digital, verde e amarelo.
- Inclusão de logo, favicon e ícones institucionais em `assets/brand`.
- Tema claro Rumo como padrão, mantendo alternância para tema escuro.
- Redução geral de escala visual para caber mais conteúdo na tela.
- Cards, botões, filtros, formulário, indicadores, flashcards e simulado mais compactos.
- Topbar e tela inicial redesenhadas com o mesmo padrão dos outros sites.
- Cache bust em CSS/JS para facilitar atualização no GitHub Pages.

## Preservado

- Conteúdo dos flashcards.
- Lógica do mini curso.
- Cadastro do fiscal.
- Simulado, correção, parecer final e impressão/PDF.
- Uso de `localStorage` já existente.
